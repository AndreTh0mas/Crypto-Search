# Setup (local host)

-  **Clone this Repo**

 ```bash

git clone (https://github.com/AndreTh0mas/Crypto-Search)

# Change directory into repository folder

cd Crypto-Search

```


``` bash
npm install

npm start

```

  






  -  *Launch website*
 
 ```bash
open browser http://localhost:3000  (without the quotes)
```
